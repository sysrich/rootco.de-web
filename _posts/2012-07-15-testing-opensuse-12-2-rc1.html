openSUSE 12.2 Release Candidate 1 is now available - Click to read the full release announcement

After a 'less than ideal' time trying to get the earlier betas running on my Thinkpad X220, I'm pleased to report that this Release Candidate installs openSUSE 12.2 perfectly.

Some First impressions

The Grub2 Boot loader is now themed (but doesn't install that theme by default - fix already done for RC2)
It's a valiant effort but the openSUSE icon needs a little work, the font is a bit ugly, and the screen resolution detection is totally borked on my i915 video card. Oh and the progress bar is an ugly grey, I think it should be a light green (Expect to see some commits from me trying to fix these on Monday)

Libreoffice is still using my early 12.2 draft artwork that is totally wrong with the final 12.2 design (Fix already submitted and in for RC2)

Power consumption on my X220 is 'better' than stock 12.1, but way worse than with my tunings from bug 717454 . Over the next few days I'll be testing the following theoretical best kernel parameters for improving power consumption and will report here if they work

pcie_aspm=powersave i915.i915_enable_rc6=7 i915.i915_enable_fbc=1 i915.lvds_downclock=1

But basically, it's all looking pretty good! Really liking GNOME 3.4
